# ngrok Auth Sample


## Overview

`ngrok` CLI のワンライン auth サンプル


## How to use

### How to run app

- `$ npm install`

- `$ node app`

### How to OAuth with ngrok

- `$ ngrok http 8080 --oauth="github"`

- `https://xxxxx.xx.ngrok.io`

### How to get Auth-ed information

- See HTTP request headers
